# AI-Powered Workforce Scheduling Assistant

## 💡 Why this matters
Manual tracking of work hours and breaks is error-prone. This tool automates scheduling based on internal constraints.

## 🧠 Features
- Start time between 7–9 AM  
- Automatic break calculation (30–60 min)  
- End time calculation  
- Rule validation  

## 🛠 Tech Stack
- Python
- pandas

## ▶️ Run
Install dependencies:
pip install -r requirements.txt

Run:
python src/main.py

## 🌍 Real-world Impact
- Presented to management  
- Adopted across teams  
- Improved workflow efficiency
